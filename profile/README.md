## Sobre Nosotros

**Astraly** está creando la próxima generación de herramientas de inteligencia de negocios y fidelización de clientes. Nuestro ecosistema combina **Agentes de IA** de vanguardia con plataformas robustas de **Comunicación Omnicanal** para ayudar a las empresas a escalar sus operaciones sin problemas.

### Módulos Principales
- **Automata**: Agentes de IA autónomos capaces de manejar flujos de conversación complejos, agendamiento y soporte al cliente a través de APIs de WhatsApp/Meta.
- **Dashboard**: Centro de comando centralizado construido con React 19 y Vite, que proporciona análisis en tiempo real, gestión de chat y configuración del sistema.
- **Integration Hub**: Conectividad perfecta con Meta (WhatsApp Business API), asegurando alta fiabilidad en la entrega de mensajes y procesamiento de webhooks.

## Arquitectura

Utilizamos una arquitectura moderna de **Microservicios** orquestada con **Docker**:

| Componente | Stack Técnico | Rol |
|:--- |:--- |:--- |
| **Backend API** | FastAPI (Async), Python 3.13 | APIs REST de alto rendimiento y WebSockets |
| **Capa de Datos** | PostgreSQL 16 (AsyncPG), Redis | Transacciones ACID y caché en tiempo real |
| **Frontend** | React 19, Tailwind v4 | UI/UX responsiva y premium |
| **Infraestructura** | AWS EC2, Nginx, Docker Compose | Despliegue escalable en la nube |
